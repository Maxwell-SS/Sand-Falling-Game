## Controls
Press the ```Colored Buttons``` at the top right of the screen to ```Switch Elements```.

Use the ```Scroll Wheel``` to ```increase``` and ```decrease``` the brush size.

## Building
```bash
mkdir build
cd build
cmake ..
make
./SandFallingGame
```
